## How to Use

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Gustavooookefne/Recupera-oSolid.git
   ```
2. Change into the project directory:
   ```bash
   cd Recupera-oSolid
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

### Running the Project
To run the main application, execute:
```bash
python module1/file1.py
```

### Running Tests
To run the tests, use:
```bash
pytest tests/
```
